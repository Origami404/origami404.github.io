Arch LightDM 配置

简要名词介绍

  • WM: Windows Manager, 用于登录之后管理窗口的玩意
  • DM: Display Manager, 提供登录之前跟登录界面
  • Greeter: 用来提供登录界面, 依赖于特定的DM
