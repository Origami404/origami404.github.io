About

 2019-07-20  302 字  自留地 非技术 

大学生, 2021 年起就读于 HITsz. 是 HITsz Linux User Group 的成员, 偶尔也会在 HITsz LUG wiki 上写点东西.

初中出于无聊学了 C++, 高中以打 OI 的名义彻底接触了 CS 的方方面面. 比起研究者更像工程师. 打 OI 心累了, 不打 ACM.

最近作为兴趣爱好主要在学编译原理啊, 类型论啊, 函数式编程之类的 PL 方向的内容.

支持自由软件运动, 但可惜没有足够大的毅力能做到多用自由软件.

你可以在 Telegram 上找到我, 也可以在 GitHub / 发 email.

本博客内容如无特殊说明, 均使用 CC BY-NC-SA 4.0 协议发布.

我在用的软件:

  • 编辑器: VSCode 一把梭
  • 各种各样软件的 theme: Nord
  • 操作系统: Windows 11 + WSL2-Arch (我用 Win, 我忏悔 qwq)
  • 曾经用过的 Linux distro (大体按时间顺序): Ubuntu, Deepin, Manjaro, Arch, Fedora
  • 密码管理器: 自建 Bitwarden
  • RSS 服务端与阅读器: Miniflux + Readably (曾经使用 TT-RSS + FeedMe)

经常看的频道/人:

  • LTT(Linus Tech Tip)
  • CppCon
  • 红警 HKB08
  • Minato Aqua
