变分学基本定理

在试卷现实生活中, 总是会碰到一些泛函上的最优化问题. 它们大多可以归为以下形式:

在函数集合

{x(t) ∣ x(t1)=y1,x(t2)=y2,x(k)存在}\left\{ x(t)\ | \ x(t_1) = y_1, x(t_2) = y_2, x^{(k)} 存在 \right\} {x(t) ∣ x(t1​)=y1​,x(t2​)=y2​,x(k)存在}

中, 求一个函数x(t)x(t)x(t)使得泛函I[x(⋅)]I[x(\centerdot)]I[x(⋅)]的积分:

S=∫t1t2I[x,x′,t] dxS = \int_{t_1}^{t_2} I[x, x', t]\ \mathrm{d}x S=∫t1​t2​​I[x,x′,t] dx

有极值.

我们知道, 一个连续函数x(t)x(t)x(t)的在某个值ttt有极值的必要条件是x′(t)=0x'(t) = 0x′(t)=0. 我们来形象而不严谨地回顾一下这是怎么得到的.

假设ttt使得x(t)x(t)x(t)取得极值, 那么在ttt附近任意取一个足够小的dt\mathrm{d}tdt, 根据极值条件, 我们有

dx<0dx<0 dx<0

同时, 因为

dx=dxdtdt\mathrm{d}x=\frac{\mathrm{d}x}{\mathrm{d}t} dt dx=dtdx​dt

此时由ttt的随意性(能大于000也能小于000), 那么只有dx/dt=0dx/dt=0dx/dt=0才能使得不等式成立(同时我们也能得到dx=0dx=0dx=0). 更严谨的证明可以看这里:维基:费马引理

那么对于泛函, 这个必要条件是什么呢?

在微积分中, 我们的一般思考方法是取极值附近的一个微元dx\mathrm{d}xdx, 然后利用函数的连续性去取得极值的必要条件, 在泛函中我们能否也去考虑一个极值函数附近的某种"微元", 即对于对应极值函数的一个小小的偏移函数, 借此来研究极值函数符合的条件呢?

我们来(随便)构造一个函数η(t)\mathrm{\eta}(t)η(t), 令其连续且存在k次导数, 并且还要在两端点为0, 不妨令:

η(t)=(t−t1)2k(t−t2)2k\eta(t) = (t - t_1)^{2k} (t - t_2)^{2k} η(t)=(t−t1​)2k(t−t2​)2k

let (k=1,t1=1,t2=2)let\ (k=1, t_1=1, t_2=2) let (k=1,t1​=1,t2​=2)

显然它符合我们的要求, 并且图像看起来就像肿了一个小包, 也挺符合我们对"偏移函数"的认知(大概?)

有了偏移函数还不够, 我们在泛函中没办法搞到像在实数集一样的序性质, 也就没办法用费马引理的老路子, 那咋办啊?

既然我们都已经有偏移函数了, 不妨引入一个实数α\alphaα, 它能控制"偏移的量", 这就把泛函的偏移归结为实数的偏移,然后我们就能用费马引理愉快(?)地玩耍了.

我们先随便找一个函数x0x_0x0​, 并令它偏移一下变成x=x0+αηx = x_0 + \alpha \etax=x0​+αη, 然后我们的SSS就从一个关于xxx的泛函变成了一个关于α\alphaα的函数:

S(α)=∫t1t2I[x0+αη,x0′+αη,t] dtS(\alpha) = \int_{t_1}^{t_2} I[x_0 + \alpha \mathrm{\eta}, x_0' + \alpha \mathrm{\eta}, t]\ \mathrm{d}t S(α)=∫t1​t2​​I[x0​+αη,x0′​+αη,t] dt

当这个函数S(α)S(\alpha)S(α)有极值时我们必有: dS/dα=0\mathrm{d}S/\mathrm{d}\alpha = 0dS/dα=0. 我们对α\alphaα求导, 有:

dSdα=∫t1t2ddαI[x0+αη,x0′+αη,t] dt\frac{\mathrm{d}S}{\mathrm{d}\alpha} = \int_{t_1}^{t_2} \frac{\mathrm{d}}{\mathrm{d}\alpha} I[x_0 + \alpha \mathrm{\eta}, x_0' + \alpha \mathrm{\eta}, t] \ \mathrm{d}t dαdS​=∫t1​t2​​dαd​I[x0​+αη,x0′​+αη,t] dt

利用多元函数的全微分, 有:

dIdα=∂I∂xdxdα+∂I∂x′dx′dα+∂I∂tdtdα\frac{\mathrm{d}I}{\mathrm{d}\alpha} = \frac{\partial I}{\partial x} \frac{\mathrm{d}x}{\mathrm{d}\alpha} + \frac{\partial I}{\partial x'} \frac{\mathrm{d}x'}{\mathrm{d}\alpha} + \frac{\partial I}{\partial t} \frac{\mathrm{d}t}{\mathrm{d}\alpha} dαdI​=∂x∂I​dαdx​+∂x′∂I​dαdx′​+∂t∂I​dαdt​

注意到最后一项中ttt并不依赖于α\alphaα, 所以原式化为:

dSdα=∫t1t2(∂I∂xη+∂I∂x′η′)dt\frac{\mathrm{d} S}{\mathrm{d} \alpha} = \int_{t_1}^{t_2} \Big( \frac{\partial I}{\partial x} \eta + \frac{\partial I}{\partial x'} \eta' \Big) \mathrm{d}t dαdS​=∫t1​t2​​(∂x∂I​η+∂x′∂I​η′)dt

我们想办法把η\etaη凑出来, 由η\etaη的任意性逼着搞出某个玩意为0. 什么玩意能把η′\eta'η′变成η\etaη呢? 当然是我们可爱的分部积分. 于是我们把第二项拉出来分部积一下:

∫t1t2∂I∂x′η′=∂I∂x′η∣t1t2−∫t1t2(ddt∂I∂x′)η dt\int_{t_1}^{t_2} \frac{\partial I}{\partial x'} \eta' = \frac{\partial I}{\partial x'} \eta \Big|^{t_2}_{t_1} - \int_{t_1}^{t_2} \Big(\frac{\mathrm{d}}{\mathrm{d}t} \frac{\partial I}{\partial x'}\Big) \eta \ \mathrm{d}t ∫t1​t2​​∂x′∂I​η′=∂x′∂I​η∣∣∣​t1​t2​​−∫t1​t2​​(dtd​∂x′∂I​)η dt

代入回去, 并注意到我们在定义η\etaη时让两端点的η\etaη值为000了, 所以把分部出来的第一项略去, 有:

dSdα=∫t1t2(∂I∂x−ddt∂I∂x′)η dt\frac{\mathrm{d} S}{\mathrm{d} \alpha} = \int_{t_1}^{t_2} \Big( \frac{\partial I}{\partial x} - \frac{\mathrm{d}}{\mathrm{d}t} \frac{\partial I}{\partial x'} \Big) \eta \ \mathrm{d}t dαdS​=∫t1​t2​​(∂x∂I​−dtd​∂x′∂I​)η dt

如果此时的函数xxx令SSS有了极值的话, 我们就会有dS/dα=0\mathrm{d}S/\mathrm{d}\alpha = 0dS/dα=0, 即:

∫t1t2(∂I∂x−ddt∂I∂x′)η dt=0\int_{t_1}^{t_2} \Big( \frac{\partial I}{\partial x} - \frac{\mathrm{d}}{\mathrm{d}t} \frac{\partial I}{\partial x'} \Big) \eta \ \mathrm{d}t = 0 ∫t1​t2​​(∂x∂I​−dtd​∂x′∂I​)η dt=0

由η\etaη的任意性, 我们就必有:

∂I∂x−ddt∂I∂x′=0\frac{\partial I}{\partial x} - \frac{\mathrm{d}}{\mathrm{d}t} \frac{\partial I}{\partial x'} = 0 ∂x∂I​−dtd​∂x′∂I​=0

综上, 当函数x使得

S=∫t1t2I[x,x′,t] dxS = \int_{t_1}^{t_2} I[x, x', t]\ \mathrm{d}x S=∫t1​t2​​I[x,x′,t] dx

有极值时, 此函数必然满足:

∂I∂x−ddt∂I∂x′=0\frac{\partial I}{\partial x} - \frac{\mathrm{d}}{\mathrm{d}t} \frac{\partial I}{\partial x'} = 0 ∂x∂I​−dtd​∂x′∂I​=0

此条件称为欧拉-拉格朗日方程.
